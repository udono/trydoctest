.. _product_template_predicate_category:

Eigenschaftskategorie
=====================
 :Modellname: ``product.template.predicate_category``
 :Basismodul: :ref:`product_predicate_category_module`
 :Menü:
        :menuselection:`Artikel --> Eigenschaften`
        :menuselection:`Artikel --> Eigenschaften --> Eigenschaftskategorien`

Das Modell *"Eigenschaftskategorie"* stellt das Konzept von
kategorisierbaren Eigenschaften von Artikeln zur Verfügung.
Ein Artikel kann über mehrere Eigenschaften verfügen.
Eigenschaften lassen sich hierarchisch als :ref:`general_concept_tree_model`
strukturieren.

.. _product_template_predicate_category_fields:

Felder
------

.. _product_template_predicate_category_active_field:

:guilabel:`Aktiv`:
    :Feldname: ``active``
    :Feldtyp: :ref:`field_type_boolean`

    Aktivieren oder Deaktivieren des Datensatzes für die weitere
    Verwendung. Siehe auch Abschnitt
    :ref:`general_concept_active_field`.


.. _product_template_predicate_category_name_field:

:guilabel:`Name`:
    :Feldname: ``name``
    :Feldtyp: :ref:`field_type_char`
    :Eigenschaft: :ref:`required`

    Der Name der Eigenschaft Kategorie.


.. _product_template_predicate_category_parent_field:

:guilabel:`Übergeordnet (Eigenschaftskategorie)`:
    :Feldname: ``parent``
    :Feldtyp: :ref:`field_type_many2one`

    Der übergeordnete Datensatz in einer Baumstruktur. Siehe auch
    Abschnitt :ref:`general_concept_tree_model`.


.. _product_template_predicate_category_children_field:

:guilabel:`Untergeordnet (Eigenschaftskategorie)`:
    :Feldname: ``children``
    :Feldtyp: :ref:`field_type_one2many`
    :Zielmodell: :ref:`product_template_predicate_category`

    Untergeordnet (Eigenschaftskategorie)


Zugriffsberechtigungen
----------------------
Die folgenden Zugriffsberechtigungen für Benutzer gelten für das Modell:

Mitglieder *ohne* Gruppe
........................
 :Lesen: erlaubt
 :Schreiben: verboten
 :Erstellen: verboten
 :Löschen: verboten

Mitglieder in Gruppe *Artikel Administration*
.............................................
 :Lesen: erlaubt
 :Schreiben: erlaubt
 :Erstellen: erlaubt
 :Löschen: erlaubt
